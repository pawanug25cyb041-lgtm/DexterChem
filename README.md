# 🔥 Smart Boiler Simulation

## 📌 Project Overview

This project simulates a smart boiler system using temperature and water level sensors.

## ⚙️ Components Used

* Temperature Sensor
* Water Level Sensor
* Relay Module
* Buzzer

## 💡 Working

* If temperature exceeds limit → Relay OFF
* If water level is LOW → Buzzer ON
